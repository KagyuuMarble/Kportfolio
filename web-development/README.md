# Web Development

Front-end exercises using HTML, CSS, and vanilla JavaScript.

**Author:** Kevin Kurpias Rodrigues

## Projects

### CSS Grid Layout — [`css-grid/`](css-grid/)

A themed landing page built to practice **CSS Grid** layout, custom web fonts (`Monocraft`), scrolling text animations, and multi-column responsive design.

- **Entry point:** `aulagrid.html`
- **Assets:** background images, GIFs, custom font files

### DOM Video Player — [`dom-video-player/`](dom-video-player/)

A custom **HTML5 video player** implemented without third-party libraries. Uses the DOM API and media events for:

- Play / pause toggle
- Seekable progress bar with time display
- Volume control and mute
- Fullscreen mode

- **Entry point:** `pagina.html`

### Intro to JavaScript — [`intro-javascript/`](intro-javascript/)

A **countdown timer** exercise covering basic JavaScript: input parsing (`MM:SS`), `setInterval` / `clearInterval`, and button state management.

- **Entry point:** `pagina.html`

## Run

Open any project's HTML file directly in a browser — no build step required.

```bash
# Example (Linux)
xdg-open css-grid/aulagrid.html
```
